wap element

Wireless access point(WAP) describes a single access point

bssid element

Basic Service Set Identifiers represent the MAC address of the access point

ssid element

Service set identifier(SSID) represents the network name

rssi element

Received signal strength signal indication(RSSI) represents he detected signal level in dBm

capabilities element

Describes the authentication, key management, and encryption schemes supported by the access point

frequency element

The freqeuncy of the channel over which the client is communicating with the access point.
